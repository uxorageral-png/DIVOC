

## Update Favicon to DIVOC Logo

Copy the uploaded logo image to the public directory and update `index.html` to reference it as the favicon, replacing the current external URL.

2. Update `index.html` to replace the current `<link rel="icon">` tag with a reference to `/favicon.jpg`
